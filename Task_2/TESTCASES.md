# TESTCASES.md
Тест-кейсы для API микросервиса Avito (тестовое задание).  
Используемый sellerID для тестов: **784512**

---

# 1. Создать объявление — POST /api/1/item

## Позитивные тесты

| ID | Название | Предусловия | Шаги | Ожидаемый результат |
|----|----------|--------------|------|---------------------|
| TC-01 | Создание объявления с корректными данными | sellerID=784512 | POST с полями sellerID, name, price, statistics | Код 200. Возвращается объект с id, sellerId, name, price, statistics, createdAt |
| TC-02 | Создание объявления с price=0 | — | POST с price=0 | Код 200. price=0 сохранён |
| TC-03 | Создание объявления с нулевой статистикой | — | statistics: {likes:0, viewCount:0, contacts:0} | Код 200. Поля статистики = 0 |
| TC-04 | Создание двух объявлений подряд | — | POST → POST | Оба объявления создаются, id разные |

## Негативные тесты

| ID | Название | Шаги | Ожидаемый результат |
|----|----------|------|---------------------|
| TC-05 | Создание без sellerID | Убрать sellerID | Код 400 |
| TC-06 | sellerID строка | sellerID: "abc" | Код 400 |
| TC-07 | Создание без name | убрать name | Код 400 |
| TC-08 | price < 0 | price: -1 | Код 400 |
| TC-09 | Некорректный тип статистики | likes: "test" | Код 400 |
| TC-10 | name > 255 символов | name=256 символов | Код 400 |
